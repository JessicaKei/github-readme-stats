# github-readme-stats

Self-hosted deployment of GitHub README Stats used for Jessica Kei profile and repositories.  

Based on:  
[https://github.com/anuraghazra/github-readme-stats](https://github.com/anuraghazra/github-readme-stats)  

See original documentation:  
[UPSTREAM_README.md](./UPSTREAM_README.md)  

<br />

## Purpose

Custom deployment for:  
- profile statistics  
- language cards  
- repository activity widgets  
- theme customization  
- caching control  
- uptime stability  

<br />

## Deployment

Hosted on **[Vercel](https://vercel.com)**.  
[https://jeskei-github-stats.vercel.app](https://jeskei-github-stats.vercel.app)
